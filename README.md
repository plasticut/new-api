new-api
=======

__Startup__:
```shell
npm install -g grunt-cli
npm install
npm start
```

__Components__:
* express https://github.com/strongloop/express
* ORM http://dresende.github.io/node-orm2/
* CRUD with express-crud https://github.com/jsdevel/node-express-crud
